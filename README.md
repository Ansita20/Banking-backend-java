# Online Banking System

This is a full-stack online banking web application.

## Tech Stack

### Backend
- **Framework:** Spring Boot (v2.7.8)
- **Language:** Java 11
- **Security & Auth:** Spring Security, JSON Web Tokens (JWT)
- **Database / ORM:** Spring Data JPA, Hibernate, MySQL, H2
- **Mailing:** Spring Mail (for OTP email verification)
- **Utilities:** Project Lombok

### Frontend
- **Framework:** React (v18.2.0)
- **State Management:** Redux Toolkit
- **Routing:** React Router DOM (v6)
- **Styling:** Tailwind CSS, Flowbite React
- **UI Components:** PrimeReact, PrimeIcons, React Icons
- **HTTP Client:** Axios
- **Notifications:** React Hot Toast

---

## Features

- **User Auth & Verification:** JWT-protected user registration and login, including OTP email verification.
- **Account Management:** Apply for and open saving/checking/current accounts and view balances.
- **Fund Transfers:** Send money directly to other bank accounts.
- **Beneficiary Management:** Add, list, and delete beneficiaries for transactions.
- **Deposits:** Create and track fixed/recurring deposits.
- **Transaction History:** View transaction history and statements.
- **Admin Dashboard:** Admin panel to approve or reject account opening requests and manage users (block/unblock).
